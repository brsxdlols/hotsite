# Portfólio Profissional | Bruno Fontes

Landing page institucional criada para apresentar o perfil técnico de **Bruno Fontes**, especialista em redes e infraestrutura para provedores de internet (ISP), com foco em autoridade profissional, clareza de informação e visual corporativo moderno.

## Sobre o projeto

Este projeto foi desenvolvido como uma página única (one page) para destacar experiência real de mercado em telecom, redes, cloud e sistemas para ISP.

A proposta visual prioriza:

- Organização e leitura rápida
- Hierarquia clara de informações
- Identidade técnica e corporativa
- Código leve, simples e fácil de manter
- Responsividade para desktop e mobile

## Estrutura da landing page

As seções implementadas são:

- Hero (apresentação principal, headline e indicadores técnicos)
- Sobre (resumo da trajetória profissional)
- Especialidades (áreas de atuação)
- Tecnologias e Ferramentas (stack por categoria)
- Experiência Profissional (timeline)
- Certificações
- Diferenciais
- Contato (links + formulário base)

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (não obrigatório neste layout)

Sem frameworks pesados, mantendo simplicidade e performance.

## Como executar localmente

Você pode abrir o arquivo `index.html` diretamente no navegador.

Se preferir rodar com servidor local:

```bash
python -m http.server 5500
```

Depois acesse:

`http://localhost:5500`

## Personalização rápida

Para adaptar o projeto para publicação final:

- Atualize e-mail, LinkedIn e WhatsApp na seção de contato em `index.html`
- Ajuste textos estratégicos conforme necessidade comercial
- Substitua a foto em `assets/images/` se necessário

## Objetivo profissional da página

Comunicar Bruno Fontes como especialista em estruturar, otimizar e escalar redes de provedores, garantindo estabilidade, segurança e performance com foco em crescimento sustentável.
